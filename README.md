# mi-primer-repositorio
## Lo que aprendí

- Comprendí cómo crear archivos y carpetas directamente desde la interfaz web de GitHub.
- Aprendí a estructurar un repositorio de forma ordenada para mantener una mejor organización del proyecto.
- Practiqué la creación de archivos CSV y comprendí su utilidad para almacenar datos simples de manera tabular.
- Reforcé el uso de commits y la importancia de documentar cada cambio realizado.
- Entendí cómo escribir secciones básicas para un README con el fin de hacer el proyecto más claro y profesional.
